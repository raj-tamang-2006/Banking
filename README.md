# 🏦 Daddy's Money Bank

A modern and responsive banking web application built using **HTML**, **CSS**, and **Vanilla JavaScript**. The project simulates core banking functionalities such as user authentication, account management, money transfers, transaction history, and profile management through a clean and premium user interface.

> ⚠️ This project is intended for educational and demonstration purposes only. It is not designed for production banking environments.

---

## 📋 Overview

Daddy's Money Bank is a client-side banking simulation that provides users with a realistic online banking experience. The application allows users to create accounts, log in securely, manage balances, transfer funds, track transactions, and update profile information.

All user data and transactions are stored locally using the browser's **Local Storage**, eliminating the need for a backend server while demonstrating fundamental banking system concepts.
